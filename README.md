# Study Focus Dashboard

This project is a 24-hour routine analysis dashboard built using Streamlit. It helps in understanding how daily time is distributed across different activities such as study, other work, mobile usage, sleep, and entertainment.

The main objective of this project is to provide a simple and interactive way to evaluate productivity and focus level based on daily habits.

## Features

- Input-based daily routine tracking
- Automatic calculation of focus score
- Visualization using pie chart, bar chart, and radar chart
- Identification of distraction time and productive time
- Smart suggestions based on user input
- Clean and interactive dashboard interface

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## How It Works

The user enters the number of hours spent on different activities in a day. Based on this input, the system calculates a focus score using predefined logic. The dashboard then displays the data in visual formats and provides insights to help improve productivity.

## How to Run the Project

1. Install the required libraries:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

## Project Purpose

This project was developed as part of academic learning to understand data visualization, user interface design, and basic logic building using Python.

It can be further extended with features like weekly tracking, data storage, or AI-based recommendations.

## Conclusion

The Study Focus Dashboard provides a simple way to monitor daily routines and identify areas where time can be better managed. It helps users become more aware of their habits and encourages better planning of their day.
